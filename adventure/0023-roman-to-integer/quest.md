# 🗡️ Quest #0023: roman-to-integer

Given a Roman numeral, convert it to an integer.

**Input format**
- One line: string representing the Roman numeral

**Output format**
- One line: integer value

**Rules & clarifications**
- Valid symbols: I, V, X, L, C, D, M

## 📋 Details  
**Difficulty:** 🟡 Medium

**Input Example:**  
```
III
```

**Output Example:**  
```
3
```

**Constraints:**  
- 1 ≤ value ≤ 3999
