# 🗡️ Quest #0012: Sort Descending

Read an integer n, then read n integers and print them in descending order (largest to smallest).

## 📋 Details  
**Difficulty:** 🟢 Easy  

**Input Example:**  
```
5
3 9 1 7 4
```

**Output Example:**  
```
9 7 4 3 1
```

**Constraints:**  
- 1 <= n <= 10^5  
- -10^9 <= numbers <= 10^9
