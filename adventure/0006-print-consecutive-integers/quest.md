# 🗡️ Quest #0006: Print Consecutive Integers

Given an integer n, print the numbers from 1 to n consecutively on the same line without any spaces or separators.

## 📋 Details  
**Difficulty:** 🟢 Easy  

**Input Example:**  
```
3
```

**Output Example:**  
```
123
```

**Constraints:**  
- 1 <= n <= 150
