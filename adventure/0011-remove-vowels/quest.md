# 🗡️ Quest #0011: Remove Vowels

Read a string and remove all vowels (a, e, i, o, u — both uppercase and lowercase) from it.  
Print the resulting string.

## 📋 Details  
**Difficulty:** 🟢 Easy  

**Input Example:**  
```
hello world
```

**Output Example:**  
```
hll wrld
```

**Constraints:**  
- 1 <= len(s) <= 10^5
