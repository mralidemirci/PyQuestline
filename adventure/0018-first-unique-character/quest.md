# 🗡️ Quest #0018: First Unique Character

Read a string and print the first character that does not repeat.  
If all characters repeat, print `"None"`.

## 📋 Details  
**Difficulty:** 🟡 Medium  

**Input Example:**  
```
swiss
```

**Output Example:**  
```
w
```

**Constraints:**  
- 1 <= len(s) <= 10^5  
- Case-sensitive
